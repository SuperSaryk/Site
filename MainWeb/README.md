# MainWeb - ASP.NET Core Web Application

A modern, responsive web application built using the ASP.NET Core framework. This project serves as a foundational web platform, demonstrating routing, server-side rendering, and user interface design.

## 🚀 Key Features

* **Bilingual Support (Localization):** Built-in toggles to switch the application interface between English and Polish dynamically.
* **Structured Navigation:** Includes a fully functional navigation bar with routing to multiple views (Home, Privacy, Game, Category).
* **Search Functionality:** Integrated search bar placeholder for querying site content.
* **Modern UI/UX:** Clean, dark-themed interface built for readability and a smooth user experience.

## 🛠️ Built With

* **C# / .NET:** The core backend framework powering the application logic and server routing.
* **ASP.NET Core:** For building the web UI and handling HTTP requests.
* **HTML5 / CSS3 / JavaScript:** For frontend structure, styling, and interactive elements (language toggles, dropdowns).

## 📦 Getting Started

### Prerequisites
* [.NET SDK](https://dotnet.microsoft.com/download) installed on your machine.
* An IDE like Visual Studio, Visual Studio Code, or Rider.
